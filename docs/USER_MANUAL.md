# Manual de Usuario de SegB 🐝

**SegB** es un bot de Telegram que analiza automáticamente las fotografías de los panales de tus colmenas. Te ayuda a calcular:

- 🍯 **Cantidad de miel operculada** (en cm² y porcentaje)
- 🔨 **Superficie de celdas construidas**
- 📏 **Escala automática** (solo necesitas dar las dimensiones del cuadro una vez)

---

## 📋 Requisitos previos

- Tener **Telegram** instalado en tu teléfono 
  - [Android](https://play.google.com/store/apps/details?id=org.telegram.messenger)
  - [iOS](https://apps.apple.com/app/telegram/id686449807)
- Una **cuenta de Telegram** activa
- Acceso a internet para enviar las fotos

---

## 🚀 Primeros pasos

### 1. Buscar el bot

Abre Telegram y busca en la lupa: **`@abejagraficabot`**

### 2. Iniciar conversación

Presiona **"Iniciar"** o envía el comando `/start`

### 3. Solicitar registro

Envía el comando: **`/login`**

### 4. Esperar aprobación

El administrador del sistema recibirá tu solicitud y te habilitará. Te llegará un mensaje de confirmación.
